# MiniGram

MiniGram is a simple Instagram-like backend project built using Node.js, Express, and EJS.

## Live Demo
https://minigram-spsn.onrender.com

## Tech Stack
- Node.js
- Express.js
- EJS
- UUID
- Method-Override


## Features
- Create Profile
- View all profiles
- View profile in detail
- Edit profile
- Delete profile
- RESTFul routing
  
## Installation (Run Locally)
1. Clone the repository
2. Install dependencies

npm install

3. Start the server

npm start

4. Open in browser:

http://localhost:8080


## Note

Data is stored in memory, so it resets when server restarts.
